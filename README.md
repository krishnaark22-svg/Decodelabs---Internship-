# 🔐 Password Strength Checker

A simple Python project to evaluate password strength based on security best practices.

## 🚀 Features
- Checks password length
- Detects uppercase, lowercase, numbers, symbols
- Classifies password as Weak / Medium / Strong
- Uses Pythonic built-in functions for efficiency
- Demonstrates secure comparison to avoid timing attacks

## 🧠 Concepts Used
- String handling
- Condition checks
- Time complexity (O(n))
- Cybersecurity basics (entropy, timing attacks)

## 📊 Rules
- Minimum 8 characters
- Must include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
